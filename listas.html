De cero a experto: Listas, Scanner y Tipos de Datos

1. Conceptos Clave

Antes de ver el código final, repasemos las herramientas que usaremos.

A. ArrayList (Listas Dinámicas)

A diferencia de un arreglo normal ([]) que tiene tamaño fijo, un ArrayList crece automáticamente cuando agregas cosas. ¡Es como una hoja de papel infinita!

import java.util.ArrayList;

// Lista de textos (String)
ArrayList<String> lista = new ArrayList<>();
lista.add("Hola"); // Agregar

B. Scanner: nextLine() vs nextInt()

Este es el error más común. nextInt() lee el número pero deja un "enter" invisible. Si luego usas nextLine(), ese "enter" se come tu lectura.

La Regla de Oro: Para evitar problemas, lee TODO como texto usando nextLine(). Si necesitas un número, conviértelo después.

C. Tipos de Datos y Conversión

Como nuestra lista guarda texto (String), si tenemos un número (int) debemos convertirlo.

  • De Texto a Número: Integer.parseInt("15")
  • De Número a Texto: String.valueOf(15)

2. Análisis de Errores

Analicemos el código original para entender qué estaba mal. ¡Aprender de los errores es la mejor forma de avanzar!

Errores Detectados:

  • Error de Scanner: Usar nextInt() para leer nombres. ¡El programa explota si escribes letras!
  • Mayúsculas: Escribir arraylist o system en minúsculas. Java no lo entiende.
  • Lógica del For: Usar i = lista.size() (asignación) en lugar de i < lista.size() (comparación).
  • Variables Duplicadas: Intentar crear una variable llamada lista dentro de un bucle cuando ya existía una lista con ese nombre afuera.

3. El Código Final (V2)

Este es el código corregido y mejorado. Pide datos de 3 alumnos, guarda todo en listas y calcula el promedio.



import java.util.ArrayList;
import java.util.Scanner;

public class ExamenCodigoV2 {
    public static void main(String[] args) {
        // 1. PREPARAR EL SCANNER
        // Creamos la herramienta 'nl' para leer del teclado.
        // System.in significa "Entrada del Sistema" (el teclado).
        Scanner nl = new Scanner(System.in);

        // 2. PREPARAR LA LISTA DE ALUMNOS
        // Aquí guardaremos a TODOS los alumnos.
        // <ArrayList<String>> significa que cada elemento de esta lista será OTRA lista (la info de un alumno).
        ArrayList<ArrayList<String>> alumnos = new ArrayList<>();
        
        // 3. VARIABLE ACUMULADORA
        // Esta variable guardará la suma de TODAS las calificaciones.
        // La iniciamos en 0 porque al principio no hemos sumado nada.
        // Si no la iniciamos, Java nos daría un error.
        int sumaCalificaciones = 0;

        // 4. CICLO FOR (Repetir 3 veces)
        // a = 1: Empezamos con el alumno 1.
        // a <= 3: Repetiremos mientras 'a' sea 3 o menos.
        // a++: Al terminar una vuelta, pasamos al siguiente alumno.
        for (int a = 1; a <= 3; a++) {
            System.out.println("=== Captura del alumno " + a + " de 3 ===");

            System.out.println("Ingresa el nombre:");
            String nombre = nl.nextLine(); // Leemos TODO el texto hasta que den Enter

            System.out.println("Ingresa el sexo:");
            String sexo = nl.nextLine();

            System.out.println("Ingresa la edad:");
            // Leemos como texto (nextLine) y convertimos a número (parseInt)
            // Esto evita el error de que se salte la siguiente pregunta.
            int edad = Integer.parseInt(nl.nextLine());

            System.out.println("Ingresa la calificacion:");
            int calificacion = Integer.parseInt(nl.nextLine());

            // --- GUARDAR DATOS DE ESTE ALUMNO ---
            // Creamos una lista temporal solo para ESTE alumno actual.
            ArrayList<String> alumno = new ArrayList<>();
            alumno.add(nombre);
            alumno.add(sexo);
            // Como la lista es de texto (String), convertimos los números a texto con String.valueOf()
            alumno.add(String.valueOf(edad));         
            alumno.add(String.valueOf(calificacion));

            // Agregamos la lista de ESTE alumno a la lista GENERAL de alumnos
            alumnos.add(alumno);
            
            // ACUMULAR CALIFICACIÓN
            // Sumamos la calificación de este alumno al total acumulado.
            // Ejemplo: Si llevamos 80 y este alumno sacó 90, ahora llevamos 170.
            sumaCalificaciones += calificacion;
        }

        // 5. MOSTRAR RESULTADOS
        System.out.println("--- Datos Guardados ---");
        // Recorremos la lista general para mostrar a cada alumno
        for (int i = 0; i < alumnos.size(); i++) {
            ArrayList<String> datos = alumnos.get(i);
            // datos.get(0) es el nombre, datos.get(3) es la calificación
            System.out.println("Alumno " + (i + 1) + ": " + datos.get(0) + " - Calif: " + datos.get(3));
        }

        // 6. CALCULAR PROMEDIO
        // Dividimos la suma total entre 3.0 (con decimal) para tener un resultado exacto.
        double promedio = sumaCalificaciones / 3.0;
        System.out.println("Promedio General: " + promedio);
        
        // 7. CERRAR SCANNER
        // Es buena práctica cerrar el scanner cuando terminamos de usarlo.
        // Esto libera la memoria y los recursos que Windows le prestó a nuestro programa.
        nl.close();
    }
}
