1. El Ciclo For

Imagina que tienes que repetir una tarea muchas veces, como saludar a 5 personas. En lugar de escribir el saludo 5 veces, usas un for.

Es como una máquina que repite instrucciones un número exacto de veces.

Estructura:

for (int i = 0; i < 5; i++) {
    // Instrucciones a repetir
    System.out.println("Hola " + i);
}
  • Inicio (int i = 0): Donde empieza el contador.
  • Condición (i < 5): Mientras esto sea verdad, sigue repitiendo.
  • Paso (i++): Aumenta el contador en 1 cada vez.

2. La Clase Scanner

El Scanner es como los oídos de tu programa. Le permite "escuchar" lo que el usuario escribe en el teclado.

Cómo se usa:

import java.util.Scanner; // 1. Importar la herramienta

// 2. Crear el objeto Scanner
Scanner scanner = new Scanner(System.in);

System.out.println("¿Cómo te llamas?");
String nombre = scanner.nextLine(); // 3. Leer texto

System.out.println("Hola " + nombre);

3. El Reto

Tu profesor quiere un programa que haga lo siguiente:

  1. Pedir los datos de 5 alumnos.
  2. Para cada alumno, pedir su nombre y su calificación.
  3. Al final, mostrar el promedio de todas las calificaciones.

4. La Solución

Aquí tienes el código completo que combina el for (para repetir 5 veces) y el Scanner (para leer los datos).

package com.escuela; // PACKAGE: Es como una carpeta para organizar tus archivos de código.

import java.util.Scanner; // Importamos la herramienta Scanner para poder leer lo que escribes en el teclado

// CLASS: Es la plantilla o molde donde escribimos nuestro programa. Todo en Java vive dentro de una clase.
public class PromedioAlumnos {
    public static void main(String[] args) {
        // 1. PREPARACIÓN
        // Creamos el objeto 'scanner' que nos servirá para escuchar al usuario
        Scanner scanner = new Scanner(System.in);
        
        // Variables para guardar datos:
        double sumaCalificaciones = 0; // Aquí iremos sumando todas las notas. Empieza en 0.
        int totalAlumnos = 5;          // Definimos que serán exactamente 5 alumnos.

        // 2. EL CICLO FOR (La repetición)
        // int i = 1; -> Empezamos contando desde el alumno 1
        // i <= totalAlumnos; -> Seguiremos repitiendo mientras 'i' sea menor o igual a 5
        // i++; -> Al terminar cada vuelta, sumamos 1 al contador 'i'
        for (int i = 1; i <= totalAlumnos; i++) {
            
            // Pedimos el nombre
            System.out.println("Ingrese el nombre del alumno " + i + ":");
            String nombre = scanner.nextLine(); // Guardamos el texto que escriba el usuario en 'nombre'

            // Pedimos la calificación
            System.out.println("Ingrese la calificación de " + nombre + ":");
            // Leemos lo que escribe el usuario y lo convertimos a número decimal (double)
            double calificacion = Double.parseDouble(scanner.nextLine());

            // 3. ACUMULACIÓN
            // Sumamos la nueva calificación al total que ya llevábamos
            sumaCalificaciones = sumaCalificaciones + calificacion;
        }

        // 4. CÁLCULO FINAL
        // El promedio es la suma de todo dividido entre la cantidad de alumnos
        double promedio = sumaCalificaciones / totalAlumnos;

        // Mostramos el resultado final en pantalla
        System.out.println("--------------------------");
        System.out.println("El promedio del grupo es: " + promedio);
        
        // Cerramos el scanner porque ya no lo usaremos más
        scanner.close();
    }
}
